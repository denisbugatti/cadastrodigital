# Protocolo de Entrada - Form Fields (25 fields)

All are text fields (/Tx):

## Empreendimento (leave blank)
- Empreendimento
- Unidade
- FAC

## Coordenador (leave blank)
- Nome_Coordenador
- CPF_Coordenador

## Cliente (fill from form response)
- Nome_Cliente → PF: nome da pessoa / PJ: nome da empresa
- E-mail_Cliente → email do formulário
- CPF_Cliente → PF: CPF / PJ: CNPJ
- Celular_Cliente → celular do formulário

## Responsáveis (leave blank)
- Nome_Diretor / CPF_Diretor
- Nome_Superintendente / CPF_Superintendente
- Nome_Gerente / CPF_Gerente
- Nome_Corretor / CPF_Corretor
